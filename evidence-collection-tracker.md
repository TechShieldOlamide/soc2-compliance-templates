## Purpose

This tracker is used to manage evidence collection throughout a SOC 2 audit, particularly during the Type II observation window. It helps ensure every required control has assigned ownership, a clear evidence type, and a status that can be reported to auditors.

---

## How to Use This Tracker

1. List every control in scope for your audit
2. Assign an evidence owner for each control (the person responsible for providing proof)
3. Define what evidence type satisfies the control (screenshot, policy document, log export, etc.)
4. Track collection status throughout the observation period
5. Flag any gaps immediately so they can be remediated before the audit closes
